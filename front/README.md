### Структура

### Архитектура: FSD

### Работа со слоями

- app - Запуск приложения: роуты, глобальные стили.
- providers - Настройка провайдеров (store)
- pages - Страницы приложения. Собираются из готовых блоков(widgets), бизнес-логики нет.
- widgets - Крупные UI-блоки страницы. Собирают фичи(features) и сущности(entities) в один компонент.
- features - Бизнес-логика и действия пользователя(функции для работы компонентов).
- entities - Основные компоненты проекта (пользователь, карточка ).
- shared - Общие вещи для всего проекта:переиспользуемые ui-компоненты, утилиты, api, картинки, шрифты.

### Сборка выполнена с помощью Vite

### Сборка проекта

- проверить установлен ли Node.js (node -v)
- проверить установлен ли npm (npm -v )
- переходите в папку front (cd front)
- выполнить команду npm install concurrently --save-dev
- выполнить команду npm run install:all

### Запуск проекта

- переходите в папку front (cd front)
- npm start команда для локального запуска приложения
